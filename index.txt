ðŸ”¹ Yeh Streamlit ka UI setup hai.

2ï¸âƒ£ File Upload System

ðŸ”¹ User CSV ya Excel file upload kar sakta hai.

3ï¸âƒ£ File Processing & Display

if uploaded_files:
    for file in uploaded_files:
        file_ext = os.path.splitext(file.name)[-1].lower()

ðŸ”¹ Har file ka extension check kiya jata hai (.csv ya .xlsx).


        if file_ext == ".csv":
            df = pd.read_csv(file)
        elif file_ext == ".xlsx":
            df = pd.read_excel(file)
        else:
            st.error(f"Unsupported file type: {file_ext}")
            continue

ðŸ”¹ CSV file ko Pandas pd.read_csv() se load kiya jata hai.
ðŸ”¹ Excel file ko pd.read_excel() se load kiya jata hai.
ðŸ”¹ Agar unsupported file ho, to error show hoti hai.


file_size_kb = len(file.getvalue()) / 1024
st.write(f"**File Name:** {file.name}")
st.write(f"**File Size:** {file_size_kb:.2f} KB")

ðŸ”¹ File ka naam aur uska size (KB me) show kiya jata hai.


st.write("Preview the Head of the Dataframe")
st.dataframe(df.head())

ðŸ”¹ File ka pehla kuch data preview ke liye dikhaya jata hai.

4ï¸âƒ£ Data Cleaning Options

st.subheader("Data Cleaning Options")
if st.checkbox(f"Clean Data for {file.name}"):

ðŸ”¹ Checkbox option diya gaya hai taake user cleaning options enable kar sake.


col1, col2 = st.columns(2)

ðŸ”¹ UI ko 2 columns me divide kiya gaya hai.


with col1:
    if st.button(f"Remove Duplicates from {file.name}", key=f"remove_dup_{file.name}"):
        df.drop_duplicates(inplace=True)
        st.write("âœ… Duplicates Removed!")
        st.dataframe(df.head())

ðŸ”¹ Duplicate rows hataane ka button diya gaya hai.
ðŸ”¹ Agar button click hota hai, to df.drop_duplicates() se duplicate rows remove hoti hain.


with col2:
    if st.button(f"Fill Missing Values for {file.name}", key=f"fill_na_{file.name}"):
        numeric_cols = df.select_dtypes(include=['number']).columns
        df[numeric_cols] = df[numeric_cols].fillna(df[numeric_cols].mean())
        st.write("âœ… Missing Values Filled!")
        st.dataframe(df.head())

ðŸ”¹ Missing values ko fill karne ka button hai.
ðŸ”¹ Numerical columns ko mean() value se fill kiya jata hai.

5ï¸âƒ£ Columns Select Karna

st.subheader("Select Columns to Convert")
columns = st.multiselect(f"Choose Columns for {file.name}", df.columns, default=df.columns)
df = df[columns]

ðŸ”¹ User select kar sakta hai ke kaunse columns rakhne hain.
ðŸ”¹ st.multiselect() se dropdown menu diya gaya hai.

6ï¸âƒ£ Data Visualization

st.subheader("Data Visualization")
if st.checkbox(f"Show Visualization for {file.name}"):
    num_cols = df.select_dtypes(include='number')
    if not num_cols.empty and num_cols.shape[1] > 1:
        st.bar_chart(num_cols.iloc[:, :2])
    else:
        st.warning("Not enough numeric data for visualization!")

ðŸ”¹ Checkbox diya gaya hai jo enable karne par graphs dikhata hai.
ðŸ”¹ Bar chart sirf numerical columns ke liye show hota hai.
ðŸ”¹ Agar numerical data nahi hai, to warning show hoti hai.

7ï¸âƒ£ File Conversion (CSV â‡„ Excel)

st.subheader("Conversion Options")
conversion_type = st.radio(f"Convert {file.name} to:", ["CSV", "Excel"], key=f"convert_{file.name}")

ðŸ”¹ Radio button diya gaya hai jisme user file ko CSV ya Excel me convert kar sakta hai.


if st.button(f"Convert {file.name}", key=f"convert_btn_{file.name}"):
    buffer = BytesIO()
    if conversion_type == "CSV":
        df.to_csv(buffer, index=False)
        file_name = file.name.replace(file_ext, ".csv")
        mime_type = "text/csv"

    elif conversion_type == "Excel":
        with pd.ExcelWriter(buffer, engine="openpyxl") as writer:
            df.to_excel(writer, index=False)
        file_name = file.name.replace(file_ext, ".xlsx")
        mime_type = "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"

    buffer.seek(0)

ðŸ”¹ File ko convert karne ka logic likha gaya hai.
ðŸ”¹ Memory me file store hoti hai (BytesIO()).


st.download_button(
    label=f"Download {file.name} as {conversion_type}",
    data=buffer,
    file_name=file_name,
    mime=mime_type
)

ðŸ”¹ Download button diya gaya hai jo converted file ko download karne ka option deta hai.

8ï¸âƒ£ Final Success Message

st.success("All files processed! ðŸŽ‰")

ðŸ”¹ Sab kuch process hone ke baad success message show hota hai.
